#### 1. Compare

Two strings are equal if they have the same length and content.

```
"aa" == "ab" #=> true

"aa" === "ab" #=> true

"aa".eql?"ab" #=> true
```


```
"abcdef" <=> "abcde" #=> 1

"abcdef" <=> "abcdef" #=> 0

"abcdef" <=> "abcdefg" #=> -1

"abcdef" <=> 1 #=> nil
```
